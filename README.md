# Run Snack, Run!
Um jogo da Serpente (Snake Game) nativo para Android desenvolvido em Kotlin. Inclui os modos Padrão, Infinito e Survival (com IA e Pathfinding), sistema de conquistas em tempo real, suporte multilíngue e UI totalmente personalizada.
